浙江大学 CAD & CG 国家重点实验室 · ZIP Lab

ZIP Lab 隶属于浙江大学 CAD & CG 国家重点实验室。我们专注于Efficient AI computing, LLM System研究，特别是大语言模型，多模态大模型, 图像视频生成模型，3D大模型的研究与应用。

📋 职位描述

我们正在寻找对AI平台开发和维护充满热情的实习生，参与我们实验室AI研究平台的建设与优化工作。你将有机会接触最前沿的AI技术，与顶尖研究团队合作，为实验室的科研工作提供强有力的技术支撑。

💻 技术要求

必备技能

  • 熟悉前后端开发流程，具备全栈开发基础
  • 掌握 Node.js 后端开发，了解 Express/Koa 等框架
  • 熟练使用 Vue.js 及其生态系统（Vue 3、Vuex/Pinia、Vue Router）
  • 熟练使用 Linux 操作系统，掌握基本的 Shell 脚本编写
  • 了解并能使用 AI 辅助编程工具（如 Claude Code、GitHub Copilot/Codex）
  • 具备良好的代码规范和文档编写能力

技术栈

Node.js Vue.js 3 TypeScript Linux Docker Git Claude Code GitHub Copilot RESTful API MongoDB/MySQL

加分项

  • 有 AI/ML 平台开发经验
  • 熟悉容器化技术（Docker、Kubernetes）
  • 了解深度学习框架（PyTorch、TensorFlow）的部署
  • 有大规模数据处理和可视化经验
  • 参与过开源项目

🌟 优先考虑

  • 研究生优先（硕士、本科在读）
